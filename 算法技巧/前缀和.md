# 前缀和
## 概念
**前缀和（Prefix Sum）** 是一种极其高效的算法技巧，它的核心思想是：**“预处理一部分数据，从而实现 $O(1)$ 时间内的区间查询。”**

简单来说，就是提前把数组从开头到当前位置的所有数字加起来，存到一个新数组里。

---

## 为什么要用前缀和？

假设有一个数组 `nums = [1, 2, 3, 4, 5]`，如果你需要求第 1 到第 3 个数的和（即 `2+3+4`）：

- **暴力法**：每次都用 `for` 循环去加。如果查询 1 万次，每次都要算半天。
    
- **前缀和法**：查表，一下就出来了。
    

---

## 怎么算前缀和？

假设原数组为 $A$，前缀和数组为 $P$。

$P[i]$ 表示从 $A[0]$ 到 $A[i-1]$ 的所有元素之和。

- **计算公式**：
    
    $$P[i] = P[i-1] + A[i-1]$$
    
    （通常为了处理边界，我们会让 $P[0] = 0$，数组长度设为 $n+1$）
    
**例子**：

原数组 `nums`: `[1, 2, 3, 4]`

前缀和 `prefix`: `[0, 1, 3, 6, 10]`

- `prefix[1] = 1`
    
- `prefix[2] = 1 + 2 = 3`
    
- `prefix[3] = 1 + 2 + 3 = 6`

当然也可以不这样做，使得P\[i] 表示A\[0]到A\[i]的所有元素之和，只不过在具体情况中需要特殊判断一下edgecase


---
## 如何查询区间和？

这是前缀和最牛的地方。如果你想求原数组中索引从 `L` 到 `R` 的区间和：

**公式：`Sum(L, R) = prefix[R + 1] - prefix[L]`**

---

## 代码实现（Java）

```java
int[] nums = {1, 2, 3, 4, 5};
int n = nums.length;
int[] prefix = new int[n + 1];

 1. 预处理：计算前缀和
for (int i = 0; i < n; i++) {
    prefix[i + 1] = prefix[i] + nums[i];
}

 2. 查询区间 [1, 3] 的和 (即 2, 3, 4)
  对应公式 prefix[R+1] - prefix[L]
int sum = prefix[4] - prefix[1]; // 10 - 1 = 9
```

---

### 5. 应用场景

- **区间求和**：频繁查询数组中某一段的和。
    
- **统计子数组**：比如求“和为 K 的子数组个数”。
    
- **二维矩阵**：二维前缀和可以快速算出矩阵中任意小矩形的数值之和。
