
# 题目

给你二叉树的根结点 `root` ，请你将它展开为一个单链表：

- 展开后的单链表应该同样使用 `TreeNode` ，其中 `right` 子指针指向链表中下一个结点，而左子指针始终为 `null` 。
- 展开后的单链表应该与二叉树 先序遍历 顺序相同。

**示例 1：**

![](https://assets.leetcode.com/uploads/2021/01/14/flaten.jpg)

**输入：** root = `[1,2,5,3,4,null,6]
**输出：**`[1,null,2,null,3,null,4,null,5,null,6]

**示例 2：**

**输入：** root = `[]
**输出：** `[]

**示例 3：**

**输入：** root = `[0]
**输出：** `[0]


# 题解

最简单的思路：前序遍历每一个节点，存储到链表中，然后再遍历链表，left置null，right指向链表中的下一个节点
这种做法，时间空间效率都低

所以采用下面的方法：
对上面的思路倒着来，右左中的后序遍历（前序是中左右），对每一个节点，每次将当前节点的右指针指向之前处理好的部分，左指针置 `null`，然后更新前驱节点prev

```java
class Solution {
    public TreeNode prev = null;
    public void flatten(TreeNode root) {
        if(root ==null) return;
        flatten(root.right);
        flatten(root.left);
        root.right = prev;
        root.left = null;
        prev = root;
    }
}
```
