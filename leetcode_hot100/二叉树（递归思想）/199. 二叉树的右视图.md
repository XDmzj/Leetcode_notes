# 题目


给定一个二叉树的 **根节点** `root`，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。

# 题解

递归二叉树，先对右子树进行递归，如果此时节点的深度和ans此时的大小相同时，将其存入ans

注意这里是大小，不是容量，大小初始为0，此时节点深度也为0（根节点），存入、递归之后，大小为1，右节点深度也为1，存入，直到右子树被遍历完毕
然后遍历左子树，左子树的前几个节点深度不够，不会存入，深度等于ans的大小之后才会开始存入

关键变量是：递归树的深度记录为depth，下一层递归时+1


```java
class Solution {
    public List<Integer> rightSideView(TreeNode root) {
        List<Integer> ans = new ArrayList<>();
        dfs(ans,root,0);
        return ans;
    }
    public void dfs(List<Integer> ans , TreeNode root, int depth){
        if( root==null) return ;
        if(depth == ans.size()){
            ans.add(root.val);
        }
        dfs(ans,root.right,depth+1);
        dfs(ans,root.left,depth+1);
    }
}
```
