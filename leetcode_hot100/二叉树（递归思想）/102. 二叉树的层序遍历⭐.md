给你二叉树的根节点 `root` ，返回其节点值的 **层序遍历** 。 （即逐层地，从左到右访问所有节点）。

**示例 1：**

![](https://assets.leetcode.com/uploads/2021/02/19/tree1.jpg)

**输入：**root = \[3,9,20,null,null,15,7]
**输出：**\[\[3],\[9,20],\[15,7]]

**示例 2：**

**输入：**root = \[1]
**输出：**\[\[1]]



# 题解

队列实现BFS

```java
class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> ans = new ArrayList<>();  返回的列表
        if(root == null) return ans ;  空树处理
        Queue<TreeNode> qu = new ArrayDeque<>(); 创建队列要用ArrayDeque
        qu.add(root);
        while(!qu.isEmpty()){
            int n = qu.size();
            List<Integer> vals = new ArrayList<>(n);
            while(n-- > 0){
                TreeNode t = qu.poll();
                vals.add(t.val);
                if(t.left!=null) qu.add(t.left);
                if(t.right!=null) qu.add(t.right);
            }
            ans.add(vals);
        }
        return ans;
    }
}
```



## 队列操作

### 添加元素 (Insertion)

|**方法**|**失败时的表现**|**使用建议**|
|---|---|---|
|**`add(e)`**|抛出 `IllegalStateException`|当你确定队列有空间，且希望空间不足时程序直接报警时使用。|
|**`offer(e)`**|返回 `false`|**推荐使用**。在有容量限制的队列（如 `ArrayBlockingQueue`）中，这更优雅，不会中断程序。|

---

### 取出并移除元素 (Removal)

|**方法**|**失败时的表现（队列为空）**|**使用建议**|
|---|---|---|
|**`remove()`**|抛出 `NoSuchElementException`|只有当你百分之百确定队列不为空时才使用。|
|**`poll()`**|返回 `null`|**推荐使用**。配合 `if (item != null)` 判断，代码更健壮。|
