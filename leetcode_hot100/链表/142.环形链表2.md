在141环形链表1的基础上找到入环节点
注意，141中快慢指针相遇的地方不一定是入环点


# 题解

~~~java
public class Solution {
    public ListNode detectCycle(ListNode head) {
        if(head == null|| head.next == null) return null;  
        ListNode fast = head;
        ListNode slow = head;
        while(true){
            fast = fast.next.next;
            slow = slow.next;
            if(fast == null || fast.next == null) return null;
            if(fast == slow) {
                while(slow != head){
                    slow =slow.next;
                    head =head.next;
                }
                return slow;
            }
        }
    }
}

~~~

## 具体数学原理
![[Pasted image 20260324174900.png]]
如果存在环，fast和slow相遇时，fast走了2x步，slow走了x步，
设环的长度为z，则fast和slow相遇时fast比slow多走了kz步（也就是多走了k圈）
2x-x = kz  ->  x = kz
设从head到入环节点的步数为y
则slow从进环到相遇走的步数为 x-y = kz-y
kz-y+y = kz对吧，就意味着在slow相遇的时候再走y步就到入口点了（变成z的整数倍了）
而head走到入环点也是y步，
所以再相遇之后slow和head同时出发，走y步就相遇再入环点了；
