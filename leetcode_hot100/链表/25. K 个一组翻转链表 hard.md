# 题目
给你链表的头节点 `head` ，每 `k` 个节点一组进行翻转，请你返回修改后的链表。

`k` 是一个正整数，它的值小于或等于链表的长度。如果节点总数不是 `k` 的整数倍，那么请将最后剩余的节点保持原有顺序。

你不能只是单纯的改变节点内部的值，而是需要实际进行节点交换。

**示例 1：**

![](https://assets.leetcode.com/uploads/2020/10/03/reverse_ex1.jpg)

**输入：** head = \[1,2,3,4,5], k = 2
**输出：**\[2,1,4,3,5]

**示例 2：**

![](https://assets.leetcode.com/uploads/2020/10/03/reverse_ex2.jpg)

**输入：** head = \[1,2,3,4,5], k = 3
**输出：**\[3,2,1,4,5]

# 题解

这道题我讲以下几点，你记一下
1. 迭代法反转链表，迭代法相比递归法过程更可控，旨在一个函数中处理链表，递归会嵌套很多层函数
2. 仔细审题，不足k个的数组就不反转了

一刷：报告长官记不住

class Solution {
    public ListNode reverseKGroup(ListNode head, int k) {
        ListNode cur = head;
        
        int count = 0;
        //从head开始取出k个节点，每取一个count++
        for(int i = 0 ;i<k;i++){
            if(cur==null) break; 
            cur = cur.next;
            count++;
        }
        if(count==k){
            ListNode node = reverse(head,cur); 
        reverse返回的结果是反转之后链表的头节点

            head.next = reverseKGroup(cur,k); 
		reverseKGroup结果是下一个k个元素的小组反转之后的头节点，这一个小组的尾部节点指向它

            return node;
        }
        return head; //如果剩下的节点数不够k个，就不反转了

    }
    private ListNode reverse(ListNode a, ListNode b) {
        ListNode prev = null;
        ListNode cur = a;
        while(cur!=b){
            ListNode tmp = cur.next;
            cur.next = prev;
            prev = cur ;
            cur = tmp;
        }
        return prev;
    }
}
