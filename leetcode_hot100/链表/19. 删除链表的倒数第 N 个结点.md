

# 题目


给你一个链表，删除链表的倒数第 `n` 个结点，并且返回链表的头结点。

**示例 1：**
![[Pasted image 20260422112659.png]]


**输入：** head = \[1,2,3,4,5], n = 2
**输出：**\[1,2,3,5]

**示例 2：**

**输入：** head = \[1], n = 1
**输出：**\[]

**示例 3：**

**输入：** head = \[1,2], n = 1
**输出：**\[1]

# 题解

思路在注释中
关键问题在引入空的头节点，才能处理删除第一个节点的情况

```java
class Solution {

    public ListNode removeNthFromEnd(ListNode head, int n) {
         1. 创建哑节点，指向链表头部
        ListNode dummy = new ListNode(0);
        dummy.next = head;
        
         2. 初始化快慢指针
        ListNode left = dummy;
        ListNode right = dummy;

         3. 快指针先走 n + 1 步
        这样快慢指针之间就隔了 n 个节点
        for (int i = 0; i <= n; i++) {
            right = right.next;
        }
        
         4. 同时移动，直到快指针指向 null
         此时 slow 正好指向“待删除节点”的前一个位置
        while (right != null) {
            left = left.next;
            right = right.next;
        }

         5. 跳过（删除）目标节点
        left.next = left.next.next;

         6. 返回哑节点的下一个节点（即真正的头节点）
        return dummy.next;
    }
}
```
