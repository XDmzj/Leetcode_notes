
# 题目

给你一个链表数组，每个链表都已经按升序排列。

请你将所有链表合并到一个升序链表中，返回合并后的链表。

**示例 1：**

**输入：** lists = `[[1,4,5],[1,3,4],[2,6]]
**输出：**`[1,1,2,3,4,4,5,6]
**解释：** 链表数组如下：
`[
  1->4->5,
  1->3->4,
  2->6
]
将它们合并到一个有序链表中得到。
1->1->2->3->4->4->5->6

**示例 2：**

**输入：**  `lists = []
**输出：**`[]

# 题解

## 分治法

分治法常用于数组类型题目（往往都是有序数组）
核心思路是：将数组对半分，然后分别处理两个部分，怎么处理呢，再对半分一次，直到不可分
然后调用处理逻辑，再一层一层`“归”`上去。
注意到，分成两个部分用的是同样的处理方式，所以想到用递归

分治参考[[分治]]

```java
class Solution {
    public ListNode mergeKLists(ListNode[] lists) {
        return mergeKLists(lists, 0, lists.length);
    }

    // 合并从 lists[i] 到 lists[j-1] 的链表
    private ListNode mergeKLists(ListNode[] lists, int i, int j) {
        int m = j - i;
        if (m == 0) {
            return null; // 注意输入的 lists 可能是空的
        }
        if (m == 1) {
            return lists[i]; // 无需合并，直接返回
        }
        ListNode left = mergeKLists(lists, i, i + m / 2); // 合并左半部分
        ListNode right = mergeKLists(lists, i + m / 2, j); // 合并右半部分
        return mergeTwoLists(left, right); // 最后把左半和右半合并
    }

    // 21. 合并两个有序链表
    private ListNode mergeTwoLists(ListNode list1, ListNode list2) {
        ListNode dummy = new ListNode(); // 用哨兵节点简化代码逻辑
        ListNode cur = dummy; // cur 指向新链表的末尾
        while (list1 != null && list2 != null) {
            if (list1.val < list2.val) {
                cur.next = list1; // 把 list1 加到新链表中
                list1 = list1.next;
            } else { // 注：相等的情况加哪个节点都是可以的
                cur.next = list2; // 把 list2 加到新链表中
                list2 = list2.next;
            }
            cur = cur.next;
        }
        cur.next = list1 != null ? list1 : list2; // 拼接剩余链表
        return dummy.next;
    }
}
```



## 最小堆方法

没见过最小堆，
最小堆维护着一个数组，但是逻辑上是完全二叉树，这个树的每个节点都比它的左右子节点小，这也就保证了根节点是最小的
最小堆有两个方法，poll()和offer()，
poll取出根节点，
offer添加一个节点，最小堆维护大小顺序


```java
class Solution {
    public ListNode mergeKLists(ListNode[] lists) {
        PriorityQueue<ListNode> pq = new PriorityQueue<>((a, b) -> a.val - b.val);
        for (ListNode head : lists) {
            if (head != null) {
                pq.offer(head); // 把所有非空链表的头节点入堆
            }
        }

        ListNode dummy = new ListNode(); // 哨兵节点，作为合并后链表头节点的前一个节点
        ListNode cur = dummy;
        while (!pq.isEmpty()) { // 循环直到堆为空
            ListNode node = pq.poll(); // 剩余节点中的最小节点
            if (node.next != null) { // 下一个节点不为空
                pq.offer(node.next); // 下一个节点有可能是最小节点，入堆
            }
            cur.next = node; // 把 node 添加到新链表的末尾
            cur = cur.next; // 准备合并下一个节点
        }
        return dummy.next; // 哨兵节点的下一个节点就是新链表的头节点
    }
}
```

最难搞的功能由最小堆实现了，所以逻辑很简单





## 合并两个有序链表

效率最低的方法
具体参考[[21.合并两个有序链表]]
```java
class Solution {
    public ListNode mergeKLists(ListNode[] lists) {
        ListNode node = new ListNode(Integer.MIN_VALUE);
        for(int i = 0 ;i < lists.length ; i++){
            node = mergeList(node , lists[i]);
        }
        return node.next;
    }

    public ListNode mergeList(ListNode l1, ListNode l2){
        ListNode dummy = new ListNode(Integer.MIN_VALUE);
        ListNode node = dummy;
        while(l1!=null && l2!=null){
            if(l1.val>l2.val){
                node.next = l2;
                l2 = l2.next;
            }else{
                node.next = l1;
                l1 = l1.next;
            }
            node = node.next;
        }
        if(l1==null){
            node.next = l2;
        }else{
            node.next = l1;
        }
        return dummy.next;
    }
}
```
