# 题目
给你一个整数数组 `nums` ，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

**子数组**是数组中的一个连续部分。

**示例 1：**

**输入：** nums = \[-2,1,-3,4,-1,2,1,-5,4]
**输出：** 6 
**解释：** 连续子数组 \[4,-1,2,1] 的和最大，为 6 。

**示例 2：**

**输入：** nums = \[1]
**输出：** 1


# 题解
这里进行了空间上的优化，但是思路是动态规划的
```java
class Solution {
    public int maxSubArray(int[] nums) {
        int pre = nums[0];
        int ans = nums[0] ;
        for(int i = 1 ; i<nums.length;i++){
                ans = Math.max(ans,Math.max(pre,0) + nums[i]);
                pre = Math.max(pre,0) + nums[i];
        }
        return ans;
    }  
}
```



## 动态规划详解
### 思路（以53为例

我们按照 DP 的三部曲来深度解析：
###  一：定义状态（`dp` 数组存什么？）

动态规划的核心是把原问题拆解成子问题。
- **我们的目标**：找到全局最大的子数组和。
- **定义状态**：设 `dp[i]` 为 **以第 $i$ 个元素结尾的连续子数组的最大和**。
> **注意：** 为什么一定要规定“以 $i$ 结尾”？因为子数组必须是连续的。只有规定了结尾，我们才能在处理第 $i+1$ 个元素时，判断它是否能连接到前面的子数组上。


### 二：状态转移方程（逻辑推导）

当我们考虑第 $i$ 个元素 `nums[i]` 时，有两种选择：

1. **加入前面的队伍**：如果前面的子数组和 `dp[i-1]` 是正数，那它对我有增益，我直接加上它。即 `nums[i] + dp[i-1]`。
2. **另起炉灶**：如果前面的子数组和 `dp[i-1]` 是负数，那它只会拖累我，我不如只取我自已。即 `nums[i]`。

**状态转移方程为：**

$$dp[i] = \max(nums[i], dp[i-1] + nums[i])$$

或者更简单的理解：**如果你前面的和大于 0，我就带上你；否则我只带我自己。**

### 三：寻找最终答案与边界
边界就是起点

- **边界条件**：`dp[0] = nums[0]`（第一个元素结尾的最大和就是它自己）。
- **最终结果**：我们要找的是整个数组中最大的子数组和，所以结果不是 `dp[n-1]`，而是 **整个 `dp` 数组中的最大值**：$\max(dp[0], dp[1], ..., dp[n-1])$。



# 前缀和方法

前缀和算法技巧参考[[前缀和]]

```java
class Solution {
    public int maxSubArray(int[] nums) {
        int[] pre = new int[nums.length];
        pre[0] = nums[0];
        int ans = Integer.MIN_VALUE;
        int minpre = 0;
        for(int i  = 0 ; i< nums.length;i++){
            if(i==0){
                ans = Math.max(ans,pre[i]-minpre);
                minpre = Math.min(minpre,pre[i]);
                continue;
            }
            pre[i] = pre[i-1] +nums[i];
            ans = Math.max(ans,pre[i]-minpre);
            minpre = Math.min(minpre,pre[i]);
        }
        return ans;
    }  
}
```

代码优化版：
```java
class Solution {
    public int maxSubArray(int[] nums) {
        int pre = 0;
        int ans = Integer.MIN_VALUE;
        int minpre = 0;
        for(int x : nums){
            pre += x;
            ans = Math.max(ans,pre-minpre);
            minpre = Math.min(minpre,pre);
        }
        return ans;
    }  
}
```

如果在运算中只用到了前一项的前缀和，那就没有必要用一整个数组来存储所有前缀和，用pre实时更新当前项的前缀和即可
