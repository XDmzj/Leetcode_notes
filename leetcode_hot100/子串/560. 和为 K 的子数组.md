

# 题目

给你一个整数数组 `nums` 和一个整数 `k` ，请你统计并返回 _该数组中和为 `k` 的子数组的个数_ 。

子数组是数组中元素的连续非空序列。

**示例 1：**

**输入：** nums = \[1,1,1], k = 2
**输出：** 2

**示例 2：**

**输入：** nums = \[1,2,3], k = 3
**输出：** 2

**提示：**

- `1 <= nums.length <= 2 * 104`
- `-1000 <= nums[i] <= 1000`
- `-107 <= k <= 107`

# 题解

nums中可能有负数，所以不能用滑动窗口
用前缀和
```java
class Solution {
    public int subarraySum(int[] nums, int k) {
        Map<Integer,Integer> map = new HashMap<>();
        map.put(0,1);
        int pre = 0;
        int ans = 0 ;
        for(int num :nums){
            pre+=num;
            if(map.containsKey(pre-k)){
                ans+=map.get(pre-k);
            }
            map.put(pre,map.getOrDefault(pre,0)+1);
        }
        return ans;
    }
}
```

前缀和参考[[前缀和]]，前缀和用哈希表来存储，方便后续查找

每一个子数组的元素和可以表示为 `Sum[i,j]=pre[i]-pre[j-1]`
符合条件的`Sum=k`
即`k=pre[i]-pre[j-1]`
也就是判断前缀和中有没有`pre[i]-k = pre[j-1]`这一项

>思考：为什么不用`pre[j-1]+k = pre[i]`来判断呢
>因为j-1 < i ，判断哈希表中有没有`pre[j-1]+k`的时候，`pre[i]`还没存进去呢
