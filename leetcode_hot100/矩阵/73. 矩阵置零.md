# 题目

给定一个 `_m_ x _n_` 的矩阵，如果一个元素为 **0** ，则将其所在行和列的所有元素都设为 **0** 。请使用 原地 算法

**示例 1：**

**输入：** matrix = `[[1,1,1],[1,0,1],[1,1,1]]
**输出：**`[[1,0,1],[0,0,0],[1,0,1]]

**示例 2：**


**输入：** matrix =`[[0,1,2,0],[3,4,5,2],[1,3,1,5]]
**输出：**`[[0,0,0,0],[0,4,5,0],[0,3,1,0]]

**提示：**

- `m == matrix.length`
- `n == matrix[0].length`
- `1 <= m, n <= 200`
- `-231 <= matrix[i][j] <= 231 - 1`

# 题解

思路在注释里

```java
class Solution {
    public void setZeroes(int[][] matrix) {
        boolean col = false; //竖列
        boolean row = false; //横行
		//将首行首列的置零与否记录一下
        for(int i = 0 ; i < matrix.length ; i++){
            if(matrix[i][0] ==0) {
                col =true;
                break;
            }
        }
        for(int i = 0 ; i < matrix[0].length ; i++){
            if(matrix[0][i] ==0) {
                row =true;
                break;
            }
        }
		
		//对剩余矩阵遍历，如果为0，则将其对应的首行首列元素设置为0
        for(int i =1 ;i< matrix.length;i++){
            for(int j = 1 ; j < matrix[0].length;j++){
                if(matrix[i][j]==0){
                    matrix[i][0]=0;
                    matrix[0][j]=0;
                }
            }
        }
        //再次遍历剩余矩阵，根据首行首列的置零与否来决定一个元素是否置零
        for(int i =1 ;i< matrix.length;i++){
            for(int j = 1 ; j < matrix[0].length;j++){
                if(matrix[i][0]==0 || matrix[0][j]==0){
                    matrix[i][j]=0;
                }
            }
        }
        //根据col和row来决定首列和首行是否置零
        if(col){
            for(int i = 0;i< matrix.length;i++) matrix[i][0] =0;

        }
        if(row){
            for(int i = 0;i< matrix[0].length;i++) matrix[0][i] =0;

        }
    }
}
```


- **标记需求**：先确定第一行和第一列本身是否包含 0。
    
- **存入状态**：遍历剩余的矩阵（从第二行第二列开始），如果 `matrix[i][j] == 0`，就将对应的第一行 `matrix[0][j]` 和第一列 `matrix[i][0]` 设为 0。
    
- **批量置零**：根据第一行和第一列的标记，将对应的元素置为 0。
    
- **处理首行首列**：最后单独处理第一行和第一列。
