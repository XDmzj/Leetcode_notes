给你一个整数数组 `nums` ，判断是否存在三元组 `[nums[i], nums[j], nums[k]]` 满足 `i != j`、`i != k` 且 `j != k` ，同时还满足 `nums[i] + nums[j] + nums[k] == 0` 。请你返回所有和为 `0` 且不重复的三元组。

**注意：**答案中不可以包含重复的三元组。

**示例 1：**

**输入：**nums = \[-1,0,1,2,-1,-4]
**输出：**\[\[-1,-1,2],\[-1,0,1]]
**解释：**
nums\[0] + nums\[1] + nums\[2] = (-1) + 0 + 1 = 0 。
nums\[1] + nums\[2] + nums\[4] = 0 + 1 + (-1) = 0 。
nums\[0] + nums\[3] + nums\[4] = (-1) + 2 + (-1) = 0 。
不同的三元组是 \[-1,0,1] 和 \[-1,-1,2] 。
注意，输出的顺序和三元组的顺序并不重要。

**示例 2：**

**输入：**nums =\[0,1,1]
**输出：**[]
**解释：**唯一可能的三元组和不为 0 。

**示例 3：**

**输入：**nums = \[0,0,0]
**输出：**\[\[0,0,0]]
**解释：**唯一可能的三元组和为 0 。



# 题解
思路来源于两数之和
target  == nums[i] + nums[j]

## 主要思路是：
先对nums排序，保证能使用双指针
从nums[0] 开始遍历，看有没有符合条件的三元组，具体的做法是：以 -nums[0] 为target，剩余的元素中做两数之和的求解，直到找到符合条件的left和right。
## 去重：
### i去重
如果nums[i] == nums[i-1] ，跳过这个元素（因为不去的话，会得到一样的结果）


### left，right去重
在找到符合条件的left和right之后，判断nums[left]和nums[left++] 以及nums[right] 和 nums[right--]是否相等，进行去重；
如果不去重会怎么样，同一个i可能对应着多个答案，不去重可能会导致得到两个重复的三元组，如{0,0,0,0,0}

~~~java
Arrays.sort(nums);  
List<List<Integer>> ret = new ArrayList<>();  
int n = nums.length;  
if( nums[0] + nums[1] + nums[2] > 0) return ret;  
if(nums[n-1] + nums[n-2] + nums[n-3] < 0) return ret;  
for (int i = 0; i < n - 2; i++) {  
      
    if(i > 0 && nums[i] == nums[i-1] ) continue; //去重  
    int left = i+1;  
    int right = n-1;  
  
    while(left < right){  
        int sum = nums[i] + nums[left] + nums[right];  
  
        if(sum < 0) left++ ;  
        else if(sum > 0) right-- ;  
        else{  
            ret.add(new ArrayList<>(Arrays.asList(nums[i], nums[left], nums[right])));  
            while (left < right && nums[left] == nums[left + 1]) left++; // 左去重  
            while (left < right && nums[right] == nums[right - 1]) right--;  
            left++;  
            right--;  
        }  
    }  
}  
return ret;
~~~
