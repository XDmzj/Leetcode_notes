# 题目
中等

给你一个字符串数组，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。

**示例 1:**

**输入:** strs = \["eat", "tea", "tan", "ate", "nat", "bat"]

**输出:** \[\["bat"],\["nat","tan"],\["ate","eat","tea"]]

**解释：**

- 在 strs 中没有字符串可以通过重新排列来形成 `"bat"`。
- 字符串 `"nat"` 和 `"tan"` 是字母异位词，因为它们可以重新排列以形成彼此。
- 字符串 `"ate"` ，`"eat"` 和 `"tea"` 是字母异位词，因为它们可以重新排列以形成彼此。

**示例 2:**

**输入:** strs = \[""]

**输出:** \[\[""]]

**示例 3:**

**输入:** strs = \["a"]

**输出:** \[\["a"]]

**提示：**

- `1 <= strs.length <= 104`
- `0 <= strs[i].length <= 100`
- `strs[i]` 仅包含小写字母

## 正解
### hash存储key与strs\[i]

重点在使用熟悉 
HashMap的方法  getOrDefault()
Arrays.sort()
String  ->  char[] 
哈希表嵌套一个列表
~~~java
class Solution {

    public List<List<String>> groupAnagrams(String[] strs) {

        Map<String , List<String>> hash = new HashMap<>();

        for(int i = 0 ; i< strs.length ; i++){

            char[] arr = strs[i].toCharArray();

            Arrays.sort(arr);

            String key = new String(arr);

            List<String> list = hash.getOrDefault(key, new ArrayList<String>());

            list.add(strs[i]);

            hash.put(key, list);
        }

        return new ArrayList<List<String>> (hash.values());

    }

}
~~~

问题：
返回结果时，直接返回newArrayList<>(map.values());就行，不用创建一个新变量；
**`Arrays.sort` 没有返回值**： 你写了 `char[] arr = Arrays.sort(...)`。实际上 `Arrays.sort` 的返回类型是 `void`，它直接原地修改数组。你需要先转成数组，再排序。
