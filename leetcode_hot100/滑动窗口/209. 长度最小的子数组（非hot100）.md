# 题目
给定一个含有 `n` 个正整数的数组和一个正整数 `target` **。**

找出该数组中满足其总和大于等于 `target` 的长度最小的 **子数组** `[numsl, numsl+1, ..., numsr-1, numsr]` ，并返回其长度**。**如果不存在符合条件的子数组，返回 `0` 。

**示例 1：**

**输入：** target = 7, nums = \[2,3,1,2,4,3]
**输出：** 2
**解释：** 子数组 `[4,3]` 是该条件下的长度最小的子数组。

**示例 2：**

**输入：** target = 4, nums = \[1,4,4]
**输出：** 1

**示例 3：**

**输入：** target = 11, nums = \[1,1,1,1,1,1,1,1]
**输出：** 0

# 题解
```java
class Solution {
    public int minSubArrayLen(int target, int[] nums) {
        int sum = 0;
        int left = 0;
        int n = nums.length;
        int ret = n + 1;
        for(int right = 0 ; right < n ; right++){
            sum += nums[right]; sum存储子数组之和
            while(sum>=target){
                ret = Math.min(ret,right-left+1);先更新ret中的最短子数组长度
                sum -= nums[left]; 将子数组剪短一下，看看还符不符合，
                left++;
            }
            如果while不成立，意味着目前的子数组和小于target，继续加
        }
        return ret < n + 1 ? ret : 0;
    }
}
```

## 思路
参考[[3. 无重复字符的最长子串⭐]]

核心：找到以每个元素为右边界的符合条件的最短子数组，遍历每个元素由for循环完成，找到以该元素为右边界的最短子数组由while循环来完成

所以不必担心没有遍历到所有状况，所有符合条件的最短子数组都找到了




一刷：
滑动窗口题的关键问题是：窗口什么时候变大，什么时候缩小，搞清楚这个条件，就能写出答案
就像这道题，sum在小于target时变大，以接近target，在大于target时缩小，以找到最小的符合条件的子数组

还有，子数组内部元素必须是连续的
